API Endpoints

  • /getBooksByAuthor

    • Query Params

    author (required) - Author's name to filter by

    Returns

    A JSON array of books. Format {author:string,country:string,language:string,link:string,pages:number,title:string,year:number,genres?:string[]}

    Example

    Request: /getBookByAuthor?author=Fyodor+Dostoevsky

    Response: [ {"author": "Fyodor Dostoevsky", "country": "Russia", "language": "Russian", "link": "https://en.wikipedia.org/wiki/Crime_and_Punishment\n", "pages": 551, "title": "Crime and Punishment", "year": 1866, "genres": [ "Psychological Fiction", "Philosophical Fiction" },...]

    Request: /getBookByAuthor?author=John+Doe

    Response: {message:"Error: No books found by author John Doe"}

  • /getBooksByTitle

  • /markAsFavorite

    • Query Params

    author (required) - author of book to mark as favorite

    title (required) - title of book to mark as favorite

    Returns

    a status message indicating success or failure. format {message: string}

    Example

    Request: /markAsFavorite

    body: author=Herman+Melville&title=Moby+Dick

    Response: No Response

    Request: /markAsFavorite

    body: author=John+Doe

    Response: {message: "Error: Missing coresponding author and title"}

    Request: /markAsFavorite

    body: author=John+Doe&title=untitledk

    Response: {message: "Error: untitled by John Doe does not exist"}

  • /getGenres

    • Query Params

    none

    Returns

    An array of genres as strings. format [string,string,...]

    Example

    Request: /getGenres

    Response: ["Historical Fiction", "Poetry", "Mythology",...]

  • /addBook

    • Query Params

    author (required) string - author of book to create

    title (required) string - title of book to create

    country (optional) string - country of book to create

    language (optional) string - language of book to create

    year (optional) number - year of publication of book to create

    pages (optional) number - number of pages of book to create

    link (optional) string - wikipedia link of book to create

    genres (optional) - genres of book to create

    Returns

    a status message indicating success or failure. format {message: string}

    Example

    Request: /addBook

    body: {book:{author:John Doe, country: USA, language: English, link:wikipedia.com/new_book, pages:100, title: new book, year: 2025, genres: [thriller, mystery]}}

    Response: {message: "Book successfully added"}

    Request: /addBook

    body: {}

    Response: {message: "Book successfully added"}
